backend/app/api/main.py
────────────────────────────────────────────────────────
[fix #1]  修复 thread_id 硬编码
  - 原：thread_id="user_session_001"（所有用户共享）
  - 改：thread_id=f"{username}_{uuid.uuid4().hex[:8]}"（每次请求唯一）

[fix #2]  上传文件处理后必须删除
  - 将文件删除逻辑移入 finally 块，确保无论成功/异常都清理临时文件

[fix #5]  文件上传安全校验
  - 新增 MIME 类型白名单（jpeg/png/webp/gif），不在白名单内直接拒绝
  - 新增文件大小限制（MAX_FILE_SIZE = 10MB）
  - 使用 rsplit 安全解析扩展名，防止空扩展名崩溃

[fix #8]  异步路由中的同步 IO 改造
  - 原：shutil.copyfileobj(image.file, buffer)（同步阻塞事件循环）
  - 改：content = await image.read()（异步读取上传内容）后再写入磁盘

[fix #13] 统一 API 响应格式
  - 所有返回值改为 ApiResponse 对象
  - 成功/无效图片/异常三条路径统一使用 status + message + data 结构
  - 移除 shutil 导入（不再使用）

[bugfix]  LangGraph 返回值改为字典访问
  - full_state.error_reason → full_state.get("error_reason")
  - full_state.analysis_results → full_state.get("analysis_results")

[bugfix]  analysis_results 空值保护
  - 新增 report is None 判断，避免 None.model_dump() 崩溃
  - 为空时返回 ApiResponse(status="error", message="分析流程未返回结果，请重试")
